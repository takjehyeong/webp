**디자인 컨셉:**
- Google Forms/Typeform 스타일의 모던하고 깔끔한 디자인
- 청록색/파란색 계열의 신뢰감 있는 색상
- 카드 기반 레이아웃과 부드러운 애니메이션

**구현 기능:**
- 다양한 질문 유형 (객관식, 체크박스, 단답형, 서술형)
- 반응형 디자인
- 폼 유효성 검사
- 제출 완료 메시지

**디자인 요소:**
- Primary: 청록색 그라데이션 (180deg)
- 부드러운 카드 섀도우와 호버 효과
- 넉넉한 여백과 가독성 좋은 타이포그래피
